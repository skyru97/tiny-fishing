# Design

`Tiny Fishing` is a slim fishing mod centered on a short, readable active loop:

1. Equip a fishing rod.
2. Cast into aimed water.
3. Wait for bite feedback.
4. Reel during the bite window.
5. Receive fish, trash, or prize rewards.
6. Fill the fishdex by discovering catches.

The mod stays intentionally narrow:

- active fishing loop only
- simple fish discovery catalog only
- no XP, levels, or regional progression systems
