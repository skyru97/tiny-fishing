# Tiny Fishing 1.0

Tiny Fishing is a lightweight Hytale fishing mod focused on a short, readable loop.

## Highlights

- simple cast, wait, and reel fishing flow
- no fishing minigame
- Fishing Codex with hidden slots to fill over time
- best-quality codex borders and discovery alerts
- global prize pool with gems, ores, and eternal crop seeds
- in-world codex access with the rod

## Commands

- `/tf codex`
- `/tf reset`

## Loot Balance

- fish: `70%`
- trash: `20%`
- prize: `10%`
