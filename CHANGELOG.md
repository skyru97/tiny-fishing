# Changelog

## 1.0

- released Tiny Fishing as a focused Hytale fishing mod
- removed the fishing minigame in favor of a simple bite-and-reel loop
- added the Fishing Codex with hidden slots and best-quality border colors
- added new fish and new best quality discovery alerts
- added global prize loot with gems, ores, and eternal crop seeds
- added in-world codex access by using the rod away from water
- added `/tf codex` and `/tf reset`
- cleaned the mod down to a single rod and a minimal gameplay scope
